# Growth Stocks CAGR Calculator: Step-by-Step Guide

CAGR (Compound Annual Growth Rate) represents the smooth annual growth rate over multi-year periods. Understanding CAGR enables accurate performance tracking and comparison across different investments. This guide provides formulas, worked examples, and calculation methods for growth stock CAGR analysis.

## Understanding CAGR

CAGR converts lumpy, volatile returns into single annual growth rate. Unlike simple average return, CAGR accounts for compounding effects across multiple years.

## CAGR Formula

**CAGR = (Ending Value / Beginning Value)^(1/Number of Years) - 1**

### Simple 5-Year Example

- Beginning value: $50,000
- Ending value: $89,340
- Years: 5

**CAGR = (89,340 / 50,000)^(1/5) - 1 = (1.7868)^0.2 - 1 = 12.3% annually**

Your investment grew at 12.3% annually for 5 years.

## Step-By-Step CAGR Calculation

### Data Gathering

From brokerage statements:
1. Initial investment date
2. Initial investment amount
3. Final value date
4. Final value amount
5. Number of years between dates

Note: For multi-year periods with contributions, use modified CAGR accounting for cash flow timing.

### Calculate Number of Years

If invested Jan 1, 2019 to Dec 31, 2024:
- Years = 2024 - 2019 = 5 years

If invested mid-year, use decimal years:
- Jan 1, 2019 to July 1, 2024 = 5.5 years

### Apply CAGR Formula

Example: VTI investment
- Started: Jan 1, 2019 with $100,000
- Ended: Dec 31, 2024 with $189,050
- Years: 5.99 ≈ 6 years

**CAGR = (189,050 / 100,000)^(1/6) - 1 = (1.8905)^0.1667 - 1 = 12.4% annually**

## CAGR with Contributions (Dollar-Cost Averaging)

When making regular contributions (DCA), use Time-Weighted Return (TWR):

### Modified CAGR with DCA

Example: $50,000 starting + $500 monthly for 5 years

1. Calculate sub-period returns
2. Weight by contribution timing
3. Compound across all periods

Simpler approximate CAGR formula:

**Approximate CAGR = (Ending Value / Average Capital Employed)^(1/Years) - 1**

For above example:
- Ending value: $89,340
- Total invested: $50,000 + (500 × 60) = $80,000
- Average capital: $65,000
- Approximate CAGR = (89,340 / 65,000)^(1/5) - 1 = 6.3%

(Note: This simplified calculation is approximate; professional financial software provides precise TWR)

## FAQ

**Q: Is CAGR the same as average annual return?**
A: No. CAGR accounts for compounding. Average return can overstate actual growth.

**Q: How do I calculate CAGR with irregular contributions?**
A: Use time-weighted return (TWR) or internal rate of return (IRR) methods. Most brokers provide these calculations.

**Q: What CAGR should I target for growth stocks?**
A: Historical S&P 500: 10% annually. Growth stocks: 12-15%. QQQ tech growth: 14-18%.

**Q: Does CAGR include dividends?**
A: Yes, if reinvested. Include all dividends in ending value for accurate CAGR.

**Q: Is higher CAGR always better?**
A: Not if it requires excessive volatility. A 12% CAGR with 8% volatility beats 15% CAGR with 25% volatility.
