## Introduction

Value investing for emergency funds focuses on identifying undervalued dividend-paying stocks trading below intrinsic value, offering both capital appreciation (emergency fund growth) and income generation (faster accumulation). Unlike speculative value investing, emergency fund value strategies prioritize safety through diversification and proven dividend histories.

## Value Investing Metrics for Emergency Funds

### Screening Criteria for Safe Value Holdings

| Metric | Target Range | Safety Reason |
|--------|-------------|--------------|
| P/E Ratio | 12-18x | Not overvalued |
| Dividend Yield | 2.5-4% | Sustainable income |
| Payout Ratio | <70% | Room for growth |
| 5-Yr Dividend Growth | 5%+ annually | Inflation protection |
| Debt/Equity | <2.0x | Financial strength |
| Free Cash Flow | Positive | Dividend sustainability |

Companies meeting all criteria: Safe value holdings

## Risk Management for Value Emergency Funds

Individual stock risk still exists. Mitigation:
- Hold 6-8 individual value stocks (diversification)
- Complement with dividend ETF (SCHD)
- Maintain 10% bonds/cash
- Monitor quarterly: If dividend cut, sell immediately

## Rebalancing Value Portfolios

Semi-annual rebalancing:
- Q2 & Q4: Review valuations
- If P/E expanded significantly: Consider trimming
- If new undervalued opportunity: Add position
- Reinvest all dividend income in underweighted positions

## FAQ

Q: Is value investing riskier than index funds for emergency funds?
A: Slightly more risky:
- Index funds hold 500+ stocks (extreme diversification)
- Value portfolio holds 6-8 stocks (moderate diversification)
- Accept 2-3% additional volatility for 1-2% additional annual return

Q: How do I find value stocks?
A: Use screening tools:
- Morningstar.com: P/E, yield filters
- FinViz: Valuation screens
- Yahoo Finance: Dividend screens

Look for: P/E < 18, Yield > 2.5%, Debt < 2.0x

Q: Should I use value ETFs instead of individual stocks?
A: Yes, if time-constrained:
- VTV (Value Index): 400+ holdings
- Lower management effort
- Similar returns to individual value picking
