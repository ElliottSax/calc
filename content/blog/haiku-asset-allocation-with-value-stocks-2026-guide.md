## Understanding the Value Factor

### What Makes a Stock "Value"?

Value stocks share four characteristics:

1. **Low Price-to-Earnings (P/E):** Trading at discount to historical/market average
2. **High Dividend Yield:** 3-5%+ current yield
3. **Low Price-to-Book (P/B):** Trading below tangible asset value
4. **Established Business:** Mature, profitable, cash-flowing operations

**Typical Value Stock Profile:**

```
S&P 500 average: P/E 22, Yield 1.3%
Value stock: P/E 14, Yield 3.5%
Growth stock: P/E 45, Yield 0.2%
```

### Historical Value Factor Outperformance

Academic research (Fama-French three-factor model) documents value premium:

**50-Year Historical Data (1974-2024):**

| Period | Value Return | Growth Return | Value Premium |
|--------|------------|--------------|--------------|
| 1974-1979 | 10.2% | 6.8% | +3.4% |
| 1980-1989 | 18.5% | 14.2% | +4.3% |
| 1990-1999 | 10.8% | 19.4% | -8.6% |
| 2000-2009 | 7.2% | -5.1% | +12.3% |
| 2010-2019 | 12.1% | 16.8% | -4.7% |
| 2020-2024 | 8.9% | 12.3% | -3.4% |
| **Full 50-year** | **10.8%** | **9.1%** | **+1.7%** |

**Key insights:**
- Value outperforms in down markets (2000-2009: +12.3% advantage)
- Growth outperforms in bull markets (1990-1999: -8.6% value deficit)
- Long-term value premium: +1.7% annually
- Comes with volatility cycles

## Value Factor Persistence: Why It Works

### Academic Explanation: Fama-French Model

Value stocks outperform because they're riskier (higher financial leverage, earnings volatility).

**Formula:**
```
Expected Return = Risk-Free Rate + Beta(Market Risk Premium) + Value Premium

Value stock:
= 4.5% + 1.2(7.5%) + 1.5%
= 4.5% + 9.0% + 1.5% = 15.0%

Growth stock:
= 4.5% + 0.8(7.5%) + (-0.5%)
= 4.5% + 6.0% - 0.5% = 10.0%

Value premium: 15.0% - 10.0% = 5.0% theoretical
Realized premium: 1.7% (due to behavioral factors, market inefficiencies)
```

### Behavioral Explanation: Market Mistakes

Investors systematically misprice value stocks due to:

1. **Recency bias:** Extrapolate recent trends too far (cheap stocks seem risky)
2. **Narrative preference:** Growth stories more compelling than cheap boring stocks
3. **Momentum chasing:** Buy winners (growth), avoid losers (value)
4. **Overconfidence:** Overestimate tech company long-term dominance

This creates opportunity for disciplined value investors.

## Frequently Asked Questions

**Q: Is value investing still valid in 2026?**
A: Yes. Value premium persists across 100+ years of data, >50 countries. Varies by cycle (underperformed 2010-2019) but recovers.

**Q: What's the best value ETF?**
A: VTV (large-cap value) or SCHD (dividend-focused value) for simplicity. SCHD for income, VTV for pure factor. Both excellent, either fine.

**Q: Should I overweight value now that it's underperformed?**
A: Dangerous—that's buying after underperformance. Stick to target allocation. Rebalance when drifts occur, don't time factor rotations.

**Q: Can I combine value with other factors (dividends, momentum)?**
A: Yes. Value + dividend = SCHD. Value + momentum = harder (conflicts). Value + quality = best combination (avoids value traps).

**Q: How much volatility does value add?**
A: Small-cap value (VBR) adds ~15% annual volatility. Large-cap value (VTV) adds ~8%. Dividend value (SCHD) adds ~5%. Factor volatility is acceptable for long-term investors.

**Q: Should I rebalance between value and growth?**
A: Yes, quarterly/semi-annually. When value outperforms and reaches 12% of target 10%, trim back and buy growth. Automatically "buy low, sell high."

**Q: Is value good for retirement accounts?**
A: Excellent. High dividend funds (SCHD) in IRA avoid annual taxes. Small-cap value (VBR) in Roth maximizes tax-free growth.

**Q: Does market cap matter for value investing?**
A: Yes. Large-cap value (VTV) more stable, small-cap value (VBR) higher returns. For conservative: large-cap. For aggressive: small-cap.
